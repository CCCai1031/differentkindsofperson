# differentkindsofperson
**人物skills（人物蒸馏实验室）**
  —— 萃取思维精华，拆解表达逻辑，识别职场陷阱，在观察与分析中成长。
## 项目简介
  本仓库是一个 “人物蒸馏” 实验场 —— 我们通过观察、分析不同个体的思维方式、表达习惯与行为模式，将其核心特质 “蒸馏” 为可学习、可借鉴、可警惕的结构化内容。
  在这里，你可以：
  借鉴优秀思维：拆解高手的决策逻辑、分析框架与认知模式
  学习表达技巧：提炼高效沟通的话术结构与表达逻辑
  识别职场 PUA：通过真实案例，掌握对操控型行为的判断与应对方法
---
## 内容分类
  1. 思维方式蒸馏 🧠
    聚焦认知模型与决策逻辑，拆解 “为什么他能想到这一层”。
    核心内容：问题分析框架、决策路径、底层认知假设
    适用场景：提升结构化思维、优化决策方式
  2. 表达技巧萃取 💬
    拆解高效沟通的底层逻辑，让表达更清晰、更有说服力。
    核心内容：话术结构、语境适配策略、关键表达细节
    适用场景：职场沟通、公开演讲、书面表达
  3. 职场行为观察 ⚠️
    通过真实案例，识别职场中的操控型行为与潜在陷阱。
    核心内容：PUA 典型特征、行为模式拆解、应对思路参考
    适用场景：职场自我保护、人际边界建立

## 使用指南
  1、按需浏览：根据需求进入对应文件夹，通过文件名快速定位内容
  2、结构化阅读：每篇内容包含「核心特质」「具体表现」「可借鉴 / 警惕点」三个模块，建议重点关注
  3、实践应用：将提炼的方法尝试应用于实际场景，在反馈中优化
